# 🧮 Calculadora Simples

Projeto desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Limpar operações
- Apagar último caractere
- Interface responsiva

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como executar

Basta abrir o arquivo `index.html` no navegador.
